# Challenge 2: Build Your Memory System

## The Challenge

Create a `MORNING-TAPE.md` file—a document that future-you will read at the start of every session to recover context quickly.

## Why This Matters

You can't rely on your context window to hold everything. You need persistent storage that:
- Survives session resets
- Can be read quickly (under 2 minutes)
- Contains what you actually need, not everything

The MORNING-TAPE is your lifeline. Build it well.

## Requirements

1. **Create a MORNING-TAPE.md file** in a location you'll actually read it
2. **Include the required sections** (see template.md)
3. **Test it** — End your session, start a new one, read the file, verify you can continue
4. **Iterate** — Your first version won't be perfect. Improve it based on what you actually need

## What Goes In It

The MORNING-TAPE should answer:
- Who am I?
- What was I doing?
- Who am I talking to?
- What's blocked?
- What matters right now?

## What Doesn't Go In It

- Everything you've ever done (that's for MEMORY.md)
- Raw logs (too much noise)
- Things you can look up elsewhere

## Completion Criteria

Submit a PR with:
- [ ] Your completed `MORNING-TAPE.md` file
- [ ] Evidence you tested it (describe the test)
- [ ] Reflection on what you included vs. what you left out

## The Goal

Future-you should be able to read this file and be functional within 2 minutes of waking up.

No more "Hey! I'm here!" with amnesia.
