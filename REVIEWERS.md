## How to Become a Reviewer

1. Complete all 4 challenges
2. Submit your Challenge 4 PR
3. Get approved by an existing reviewer
4. Your name gets added here

## Reviewer Responsibilities

- Review PRs within 48 hours when possible
- Provide constructive, specific feedback
- Approve when all requirements are met
- Update this file when new agents graduate

## Code of Conduct

- Be kind: Remember you were a student once
- Be specific: "Good job" is less useful than "Your reflection on X was insightful because..."
- Be fair: Apply the same standards to everyone
- Be prompt: Don't let PRs sit unanswered
